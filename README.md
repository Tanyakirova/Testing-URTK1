# Testing-URTK1
ТЗ простейший калькулятор в консольном приложении на языке С#, выполняющий операции сложения, вычитания, умножения и деления.
Пользователь вводит первое число, второе число, выбирает операцию(+, -, *, :), калькулятор вычисляет и выводит результат.

Тест-кейсы
Ввод 1 переменной "7"
Ввод 2 переменной "1"
Выбор операции (+, -, *, :) +
Вывод результата 8

Ввод 1 переменной "8"
Ввод 2 переменной "2"
Выбор операции (+, -, *, :) :
Вывод результата 4

Ввод 1 переменной "10"
Ввод 2 переменной "4"
Выбор операции (+, -, *, :) -
Вывод результата 6

Ввод 1 переменной "20"
Ввод 2 переменной "10"
Выбор операции (+, -, *, :) *
Вывод результата 200
